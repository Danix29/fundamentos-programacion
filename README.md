## About

**Asignatura:** Fundamentos de Programación &middot; UAH GII &middot; Curso 2024-25

First programming course. Introduction to algorithmic thinking, problem decomposition and Java syntax. Covers imperative and procedural programming before transitioning to basic OOP concepts.

---

## Topics covered

| Block | Content |
|-------|---------|
| Basics | Variables, data types, operators, type casting, expressions |
| Control flow | if/else, switch, for/while/do-while loops, break/continue |
| Functions | Method definition, parameters, return values, scope, overloading |
| Arrays | 1D and 2D arrays, iteration, sorting (bubble, selection, insertion) |
| Recursion | Base cases, recursive calls, stack frames, factorial, Fibonacci |
| Strings | String manipulation, StringBuilder, common String methods |
| Intro to OOP | Classes and objects, attributes, basic constructors |
