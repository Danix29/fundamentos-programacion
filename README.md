## About

**Asignatura:** Fundamentos de Programación &middot; UAH GII &middot; Curso 2024-25 &middot; 1er Curso 1er Cuatrimestre &middot; 6 ECTS

First programming course. No prior programming knowledge assumed. The course has two parts: a theory part covering language-agnostic software design concepts and algorithmic thinking, and a lab part providing hands-on Python programming from scratch.

---

## Topics covered

| Block | Hours | Content |
|-------|-------|---------|
| Programming fundamentals | 16h | Syntax and semantics of high-level languages, variables, data types, expressions, assignments, basic I/O (`input`/`print`), selective structures (`if`/`elif`/`else`), iterative structures (`for`/`while`) |
| Modularisation | 16h | Functions, parameter passing (positional, keyword, default), return values, scope, modular decomposition of programs |
| Algorithms and methodology | 14h | Problem-solving strategies, algorithm design and properties, implementation, testing and debugging techniques |
| Fundamental data structures | 18h | Lists, tuples, sets, dictionaries — creation, indexing, slicing, iteration, comprehensions; string operations; file I/O |
